최적화

- 애플리케이션의 성능을 향상시키는 방법을 찾고 적용하는 것을 의미한다.
  빠른 로딩 시간 / 사용자 경험 / 자원의 효율적인 사용

촤적화 필요를 못느끼는 이유

- 고성능 컴퓨터와 빠른 인터넷 (개발자가 최적화 필요성을 못느낌)
- 거대한 애플을 건드려야하는 부담 (최적화를 미룬다)
- 일이 너무 바쁨 (성능 < 기능 이라 장기적으로 사용자 경험 저하)

최적화 장점
시스템 확장 / 유지 관리 비용 감소 / 사용자 경험 향상
