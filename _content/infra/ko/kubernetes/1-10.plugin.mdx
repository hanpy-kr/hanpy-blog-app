# Plugin

Plugin을 통하여 쿠버네티스 확장또한 가능합니다.

목적(저장 공간 / 네트워크 / 장치)에 따라 저러 플러그인이 존재합니다.

- CSI (Container Storage Interface)
- CNI (Container Network Interface)
- Device Plugin (ex. GPU (쿠버에서 기본적으로 지원하지 않는 하드웨어에 인직과 지원을 도와주는 플러그인이다.))
