# Nodejs 설치 및 환경변수 설정

window에서 nodejs 설치와 환경변수 설정을 하는 방법을 알아봅시다. 기본적으로 nodejs는 아래의 홈페이지에서 설치를 진행합니다.



![image](https://resource.han-py.com/blog/logs/nodejs_logo.svg)




## Nodejs 설치

홈페이지(https://nodejs.org/ko/)에 접속하여 Nodejs를 설치할 수 있습니다. 설치할 수 있는 선택 방법은 LTS버전과 현재버전으로 나누어 집니다. LTS 버전은 비교적 옛날 버전이지만, 많은 개발자들이 사용한 만큼 안정화된 버전이라고 할 수 있습니다. 따라서 현업에서는 LTS 버전을 많이 사용합니다. 현재 최신버전이 나중에는 LTS버전이 되기 때문에 혼자 프로젝트를 진행한다면 현재 버전을 받고, 같이 사용하는 프로젝트라면 LTS를 사용하면 된다.

---

## Nodejs 환경변수

설치를 했으면 sysdm.cpl을 검색(윈도우키 + R)합니다. 아래의 창이 뜨면 아래의 환경변수를 클릭합니다.



![image](https://resource.han-py.com/blog/nodejs/1.png)



위에서 기본으로 설치를 했다면 설치 위치는 C:\Program Files\nodejs 입니다. 아래와 같이 추가해 줍니다.



![image](https://resource.han-py.com/blog/nodejs/2.png)



위에서 시스템 변수 부분에서 변수를 NODE_HOME으로, 값을 위의 주소를 넣어 줍니다. 그 후에 변수가 Path인 부분을 누르고 편집을 누르면 아래와 같은 화면이 나옵니다.



![image](https://resource.han-py.com/blog/nodejs/3.png)



위의 %NODE_HOME%을 추가해 주면 됩니다. 이와 같이 추가를 하면, 위에서 변수/값 추가한 부분이 들어갑니다. 그 후에 window에서 cmd를 검색하고 아래와 같이 명령어를 실행시켜 설정을 확인해 봅시다.

```bash title="cmd" showLineNumbers
$ node -v
$ npm -v
```

실행 결과로 관련 버전이 나온다면 정상적인 설치 및 설정이 완료된 것입니다.
